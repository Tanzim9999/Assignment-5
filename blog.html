Important Questions & Answers

What is the Document Object Model (DOM)?

Dom is like a structure of the HTML, Using it we can see, edit , delete html elements easily

How do you select an element from the DOM?

We can select an element from the dom using specific ID, tag or class name. We can also use querySelector and querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

By using event delegation we can use one event listener on parent element to manage multiple child elements of it

How do you manipulate an element's attributes and styles using the DOM?

By using getAttribute(),setAttribute() and removeAttribute() and for adding or removing classes using classList we can manipulate an element's attributes and styles using the dom
